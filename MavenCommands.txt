1) mvn help:describe -Dcmd=<<PHASENAME>> ==> To list all the goals of a phase
	eg: mvn help:describe -Dcmd=compile

2) Maven local repository is a folder location on your machine. It gets created when you run any maven command for the first time. Maven local repository keeps your project's all dependencies (library jars, plugin jars etc.). When you run a Maven build, then Maven automatically downloads all the dependency jars into the local repository. It helps to avoid references to dependencies stored on remote machine every time a project is build.
	Command to run a prog on local repo is : mvn exec:java -Dexec.mainClass="<<PATH TO CLASS CONTANING MAIN>>"

3) mvn help:descrive -Dcmd=compile -Doutput=<<OUTPUT FILE NAME>> ==> To redirect output/build-log of any maven command
	eg : mvn help:describe -Dcmd=compile -Doutput=compileoutput.txt

4) mvn -o help:describe -Dcmd=install ==> -o or --offline for making maven work offline
